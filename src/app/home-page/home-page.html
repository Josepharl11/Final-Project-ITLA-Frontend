¿Qué es ITLA Maps?

ITLA Maps es una solución digital que te ayuda a navegarcon facilidad por los espacios del Instituto Tecnológico de Las Américas. Un mapa interactivo diseñado específicamente para estudiantes de nuevo ingreso y que necesitan orientarse rápidamente en el campus.

👨‍💻

Creado por estudiantes, para estudiantes

Este proyecto fue desarrollado por 2 estudiantes de la carrera de Ingeniería en Sistemas del ITLA con el objetivo de resolver un problema real: la dificultad que enfrentan estudiantes nuevos y antiguos para movilizarse por las diferentes áreas del campus.

⚙️

Stack tecnológico MEAN

Utilizamos las tecnologías más modernas: MongoDB para la base de datos, Express para el backend, Angular para el frontend, y Node.js para el servidor. Un stack completo y robusto para una experiencia fluida.

🗺️

Simplicidad y Belleza

El mapa está realizado con SVG (Scalable Vector Graphics), lo que garantiza gráficos nítidos y rápidos. Explora cada edificio, piso, aula y oficina de forma intuitiva. Cada espacio tiene su descripción para que sepas exactamente qué encontrarás en cada lugar del campus.
